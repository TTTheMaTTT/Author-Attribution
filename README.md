Здесь планируется хранить все необходимые материалы по проекту "Определение автора текста" по предемету "Интеллектуальные системы".

В папке TextInstruments предназначена для хранения всех модулей, которые понадобятся для предобработки текста. 
Остальные папки предназначены для хранения трёх различных моделей для решения задачи определения автора:
1) ClassicModel - модели, использующие такие классические методы машинного обучения, как наивный байес и SVM, работающие на различных текстовых признаках;
2) RNN - модели, основанные на рекуррентных сеть, которая проходит по эмбеддингам (слов или предложений) с различными модификациями;
3) NNLModel - языковые модели, основанные на нейронных сетях.
